# api-secretSanta
This project is an api in node js, for a secret santa project

## Installation
- docker compose up -d

## Close
- docker compose down

## Package
- express
- bcrypt
- jsonwebtoken
- mongoose
- swagger-jsdoc
- swagger-ui-express

## Documentation
API documentation on the following url : http://localhost:3003/api-docs
